About this Project

Nori-Bot

Chat Bot version of Nori, supporting commands on different platforms

Nori-Wynn

Third-party app for online RPG game "Wynncraft", providing features including player/guild statistics, leaderboard, class-builds, item-related, lootpool information, and more.

Nori-API

Database hosted on AWS for general purposes, tracking information including command usage, user activity, in-game player stats, user prompts for generative AI features, generation results from models, etc...

Nori-ToolKit

Utility tools made for various purposes, consisting of API wrappers, data analysis & visualization, testing, and some specific stuff.

Nori-Web

Web version of Nori, which happens to be the site you are on right now.
